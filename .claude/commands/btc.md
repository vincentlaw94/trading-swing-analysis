# BTC Quick Analysis

Run a quick BTC/USDC analysis showing current price, BB position, and trading bias.

## Instructions

1. Run the single analysis script:
   ```bash
   node src/analyze-once.js 4H
   ```

2. Summarize the key findings:
   - Current price and BB position
   - Trading bias (BULLISH/BEARISH/NEUTRAL)
   - Key entry and exit levels
   - Recommended action

Keep the response concise - just the essential trade setup.
